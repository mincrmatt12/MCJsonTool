# MCJsonTool

> jsons the easy way

## How to run

To run, install python requirements in requirements.txt (or make a virtualenv)
Then, run main.py

### Requirements

- Python >=3.6
- PyQt5
- PyOpenGL
- numpy
- PyGLM (if you do python 3d, check this out! it's fairly new)
- pillow
